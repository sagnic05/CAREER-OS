# 🚀 DevCareer OS

DevCareer OS is a secure, full-stack command center built to track your algorithmic progress, optimize your deployed portfolio, and scale your engineering career. 

**Live Demo:** [career-os-sable.vercel.app](https://career-os-sable.vercel.app)

---

## ✨ Features

- **Secure Authentication:** Seamless and secure login via GitHub using NextAuth.js.
- **DSA Tracking:** Monitor, log, and organize Data Structures and Algorithms progress.
- **Modern Database Architecture:** Built on PostgreSQL using the latest Prisma 7 ORM with Edge/Driver Adapters (`@prisma/adapter-pg`).
- **Cyberpunk UI:** A sleek, dark-themed user interface built with Tailwind CSS and Lucide React.
- **Fully Responsive:** Optimized for both desktop and mobile command-center viewing.

---

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router & Turbopack)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Database:** PostgreSQL
- **ORM:** [Prisma v7](https://www.prisma.io/) (with `@prisma/adapter-pg` driver adapter)
- **Authentication:** [NextAuth.js](https://next-auth.js.org/) (GitHub OAuth Provider)
- **Deployment:** [Vercel](https://vercel.com/)
